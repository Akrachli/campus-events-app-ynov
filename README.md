#  Campus Events App

Application mobile Flutter pour la gestion des événements étudiants.

## Démarrage Rapide

\`\`\`bash
flutter pub get
flutter run
\`\`\`

- ✓ UI/UX avec navigation fluide
- ✓ Base de données Firebase Firestore
- ✓ Authentification Firebase Auth
- ✓ Notifications locales
